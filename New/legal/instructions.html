How do i edit this website?

scrolling content

On Desktop in the Webflow Designer, there's no way to side scroll the page unless you're in Preview mode which doesn't help much as the content can't be edited in Preview mode, so we need to do these easy steps to see all the side scrollable content.

Step 1

Locate the element in the Navigator panel called "Camera" and select it.

Step 2

With the  "Camera"  highlighted, go to the Styles panel, and change
1. "Width"  from  "Auto"  to  "100%"
2. "Overflow"  from  "Visible" to "Auto/Scroll"

done!

Now you'll see the side scroll bar at the bottom of the Designer window, you can use this to navigate around the side-scrolling content.

overlaying content

Step 1

Locate the element in the Navigator panel called "Camera" and select it.

Step 2

With the  "Bg Loop Wrapper"  highlighted, go to the Styles panel, and change
1. "Display"  from  "Flex"  to  "None"

Step 3

Locate the element in the Navigator panel called "Infinite Text Wrapper" and select it.

Step 4

With the  "Infinite Text Wrapper"  highlighted, go to the Styles panel, and change
1. "z-index"  from  "-1"  to  "0"

done!

Now you'll be able to edit contents without any overlaying blocks!
